# wheel-of-lunch
A tool for picking where to eat

<ul>
  <li>The file you want is "Bin\Wheel-Of-Lunch-0_0_1.zip"</li>
  <li>Unzip the contents to a folder and run "Wheel-of-Lunch.bat"</li>
  <li>The program will then pick a restaurant in Prince Albert to eat at.</li>
</ul>

<ul>
<li>Change "restfile.txt" in your contents folder to change the list of restaurants.</li>
  <li>Each line must contain one restaurant.</li>
</ul>
